# ShopSphere

ShopSphere is a responsive e-commerce frontend built using React, Redux Toolkit and Tailwind CSS. Product data is fetched from the Fake Store API, allowing users to browse products, search by name, filter by category, manage a shopping cart and wishlist, and view detailed product information.

## Features
- Browse products from the Fake Store API
- Product details page
- Category filtering
- Product search
- Shopping cart with quantity controls
- Wishlist
- Persistent cart and wishlist using localStorage
- Login page with client-side validation
- Responsive design for mobile, tablet and desktop
- Loading and error states for API requests
- Banner carousel
- FAQ accordion
- Unit tests using React Testing Library and Vitest

## Tech Stack
- React
- Redux Toolkit
- React Router
- Tailwind CSS
- Vite
- React Icons
- React Testing Library
- Vitest

## Installation

Clone the repository and install the dependencies.
```
git clone https://github.com/Fe-nyx/ShopSphere-Ecom-Project.git
cd ShopSphere-Ecom-Project
npm install
```

Start the development server.
```
npm run dev
```

Create a production build.
```
npm run build
```
